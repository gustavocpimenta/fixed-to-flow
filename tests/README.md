# Tests Directory

## Visual Tests (`/visual/`)

This folder contains HTML test pages used to document and validate the design process.

**Purpose:**
- Document design iterations and experiments
- Test individual components in isolation
- Preserve design decisions and alternatives explored

**Contents:**
- Animation tests (hero, sections, services)
- Component tests (bio, footer, cookie banner, confetti)
- Layout tests (services, philosophy blocks)
- Style guide reference

**Important:** These files are kept for documentation purposes. Do not delete.
