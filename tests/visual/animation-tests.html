Summary: Recommended Animations

High Impact, Low Risk

  • #1 Accent Lines - Draw on load adds polish
  • #4 Logo Stagger - Professional reveal effect
  • #7 Card Stagger - Already partially implemented
  • #8 Floating Shapes - Subtle depth without distraction
  • #9 Button Micro-interactions - Lift + arrow slide
  • #10 Form Focus - Better feedback on inputs

Medium Priority

  • #2 Section Labels - Nice but may be over-designed
  • #6 Step Numbers - Ring expand is subtle and effective
  • #12 Link Hovers - Underline from left is clean
  • #13 FAQ Accordion - Smooth expand improves UX

Use Sparingly

  • #3 Text Highlights - Can feel gimmicky if overused
  • #5 Number Counters - Only for key stats
  • #11 Quote Marks - Subtle pop is nice but optional
  • #14 Founder Image - Personal touch but may distract

Always Include

  • Scroll Progress - Top bar showing page progress
