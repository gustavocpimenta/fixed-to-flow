## Usage Recommendations

### Test by Context:

**For cold outreach or new connections:** Use Minimal (#2, #10, #16) or Achievement (#6, #13)

**For existing clients or warm relationships:** Use Personal (#5, #11, #17) or Curious (#3, #8)

**For thought leadership emails:** Use Provocative (#1, #7, #12, #18) or Curious (#14, #20)

**For service promotion campaigns:** Use Direct (#4, #9, #15, #19)

### Rotation Strategy:

1. Pick 3-5 favorites across different categories
2. Rotate weekly or monthly to test what resonates
3. Track which signatures get the most replies or clicks (if using link tracking)
4. Note which personas/contexts respond to which styles

### Customization Notes:

- All LinkedIn links should use actual profile: linkedin.com/in/gustavocpimenta
- Calendly link can be customized: calendly.com/gustavocpimenta/[specific-meeting-type]
- Newsletter link is: fixedtoflow.substack.com
- Website is: fixedtoflow.com (or specific landing pages)
- Add tracking parameters to URLs if desired (e.g., ?utm_source=email&utm_medium=signature)

### A/B Testing Ideas:

Test pairs against each other:
- Provocative vs. Minimal (high-energy vs. understated)
- Question-based vs. Statement-based
- Service-specific vs. General positioning
- Credibility markers vs. No credentials
- Newsletter focus vs. Service focus
